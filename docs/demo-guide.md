# 演示指南

## 演示场景

### 1. RAG 知识库检索

**提示词**：
```
从知识库找 ES hybrid 检索的实现要点
```

**预期效果**：
- 回答下方出现"引用来源"折叠区域
- 展开可查看检索到的文档片段和分数
- 右侧 TracePanel 显示检索详情

### 2. 工具调用

**提示词**：
```
查一下当前时间
```

**预期效果**：
- 显示工具调用卡片
- 展开可查看入参和出参摘要
- TracePanel 显示工具调用信息

### 3. 多轮对话上下文

**提示词**（依次发送）：
```
你好
```
```
帮我分析接口延迟突增该怎么排查
```
```
具体怎么做
```

**预期效果**：
- TracePanel 显示上下文管理信息
- 策略：最近消息 + 摘要 + 用户事实
- Token 使用进度条
- Facts 数量

### 4. 文件上传

**操作步骤**：
1. 点击侧边栏 **↑ 上传按钮**
2. 拖拽或选择文件（PDF/TXT/MD）
3. 点击"上传"

**预期效果**：
- 上传进度条
- 解析状态动画（QUEUED → PARSING → INDEXED）
- 成功提示

### 5. SRE 告警分析

**操作步骤**：
1. 点击侧边栏 **活动图标**
2. 选择告警卡片

**预期效果**：
- Agent 编排时间线
- 最终 Markdown 报告

### 6. HITL 工具审批

**提示词**：
```
帮我记住这个知识：Q: 什么是 RAG？A: 检索增强生成
```

**预期效果**：
- 审批弹窗显示
- 工具名、参数
- 批准/拒绝按钮

### 7. 流式响应

**提示词**：
```
帮我分析接口延迟突增该怎么排查
```

**预期效果**：
- 逐 token 流式输出
- 实时显示生成状态
